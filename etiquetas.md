Etiquetas Relevantes

-<!DOCTYPE hmtl>: Etiqueta principal, indica que es una página de tipo HTML.

-<html>...</html>: Almacena todo el contenido de la página, incluyendo lo visual y los metadatos. Se utiliza la etiqueta lang para indicar el lenguaje de la página.

-<head>..</head>: Provee información general (metadatos) acerca del documento, incluyendo su título. Se utiliza la etiqueta profile para indicar los URIs de uno o más perfiles de metadatos, separados por un espacio en blanco.

-<title>...</title>: Almacena el titulo de la página, este aparecerá en la pestaña.

-<body>...</body>: Almacena todo el contenido de la página.

-<blockquote>...</blockquote>:Indica que el texto adjunto es una cita extendida. Se utiliza la etiqueta cite de valor url para designar un documento o mensaje fuente para la información citada.

-<q>...</q>:indica que el texto adjunto es una cita corta en línea. Se utiliza la etiqueta cite de valor url para designar un documento o mensaje fuente para la información citada.

-<span>...</span>:Agrupar elementos, texto o aplicar estilos específicos a una parte pequeña de un documento.

-<figure>...</figure>:Agrupar contenido autónomo

<figcaption>...</figcaption>: Descripción de dicho contenido, pie de foto
-<b>...</b>: Resalta texto en negrita

-<strong>...</strong>:Define texto con una fuerte importancia, seriedad o urgencia. Indica a los navegadores que el contenido es importante

-<em>...</em>:Marca texto que requiere énfasis

-<i>...</i>:representa un rango de texto que se desmarca del texto normal

-<mark>...</mark>: Representa texto que está marcado o resaltado

-<u>...</u>: Muestra texto subrayado.

-<s>...</s>:Muestra texto con un tachado o una línea a través de él

-<br>: Produce un salto de línea

-<img>: Inserta imagenes.

Atributos img

src:La URL de la imagen. Este atributo es obligatorio para el elemento <img>

align:Alineamiento de la texto respecto al contexto que la rodea.

alt:define el texto alternativo que describe la imagen

border: Anchura del borde alrededor de la imagen.

height:La altura de la imagen en píxeles

width:El ancho de la imagen en píxeles

hspace: El número de píxeles de espaciado a la izquierda y la derecha de la imagen.

longdesc:La URL como descripción de una imagen mostrada, complementa al texto de alt

-<h1> a <h6>: Encabezados o titulos.

-<p>..</p>: Define párrafo.

-<button>...</button>: Es un elemento cliqueable tipo botón. Se utilizan las etiquetas:

name: Texto informativo o titulo

id:Le da un nombre al elemento que lo diferencia de todos los demás del documento.

style:Permite especificar información de estilo

name:Asigna un nombre de control.

type:Indica el tipo de botón.Uno de los siguientes: (button|submit|reset)

-<ol>...</ol>: Representa una lista ordenada de elemento.Se utilizan las etiquetas:

start:Un entero desde el que empezar a contar para los elementos de la lista.

type:Establece el tipo de numeración(a,A,i,I,1)

reversed:Especifica que los elementos de la lista están en orden inverso

-<ul>...</ul>:Representa una lista no ordenada. Se utilizan las etiquetas title, style, id para declarar sus atributos

-<li>...</li>: Declara elementos dentro de una lista.Se utilizan las etiquetas title, style, id para declarar sus atributos

-<a>...</a>: Se utiliza para crear hipervínculos en el contenido que el usuario puede hacer clic.Se utilizan las etiquetas:

href: Define el destino del enlace (una URL, un archivo, o un ancla interna).

target: Define dónde abrir el enlace. Ej: _blank abre el enlace en una nueva pestaña.

-<table>...</table>: Almacena el contenido de una tabla. Se utilizan las etiquetas:

align:Indica como va a encontrarse alineada la tabla respecto al contenido del documento(left,center,right)

bgcolor: Define el color de fondo de la tabla

border: Define el tamaño del cuadro alrededor de la tabla

cellpadding :Define el espacio entre el contenido de una celda y su borde

width:Define el ancho de una tabla

<thead>...</thead>: Agrupa el contenido del encabezado. Categorias

<tbody>..</tbody>: Agrupa el contenido principal o cuerpo de la tabla.

<tfoot>...</tfoot>: Agrupa el contenido del pie de la tabla.

<tr>...</tr>: Define una fila dentro de la tabla. Se utilizan las etiquetas:

align: especifica como será gestionado el alineamiento horizontal del contenido de cada celda (left, center, right,justify, char)

bgcolor :Define el color de fondo de cada celda de la fila

valign :Especifica la alineación vertical del texto dentro de cada fila de las celdas de la cabecera de la tabla.(baseline, bottom, middle, top)

<td>...</td>: Datos que se ubican en la fila

<th>...</th>: Define una celda de encabezado (generalmente en negrita y centrada). Se usa para títulos de columnas o filas.

<caption>...</caption>: Añade un título o descripción a la tabla.
